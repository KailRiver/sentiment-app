# Russian Sentiment Analysis API

Spring Boot приложение для анализа тональности русских текстов с использованием простой словарной модели.

## Функциональность

- REST API для анализа тональности текста
- Поддержка русского языка
- Простая словарная модель на основе JSON
- Мониторинг через Spring Boot Actuator
- Готовность к контейнеризации и развертыванию в Kubernetes
